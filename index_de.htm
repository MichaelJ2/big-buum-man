Über Big-Buum-Man

Big-Buum-Man ist ein Multiplayer-Bomberman-Spiel für verschiedene Plattformen und besteht aus einem Server und einem Client.

Der Server und unterstützt momentan folgende Plattformen:

  • Windows
  • Linux
  • Mac OS X

Der Client unterstützt momentan folgende Plattformen:

  • Windows
  • Linux
  • Mac OS X
  • Android
