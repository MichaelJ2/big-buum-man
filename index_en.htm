About Big-Buum-Man

Big-Buum-Man is a multiplayer-Bomberman-game for different platforms and consits of a server and a client.

The server currently supports the following platforms:

  • Windows
  • Linux
  • Mac OS X

The client currently supports the following platforms:

  • Windows
  • Linux
  • Mac OS X
  • Android
